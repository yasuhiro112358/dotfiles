# zsh 起動時の読み込み順と挙動

## 目的
`.zprofile` と `.zshrc` の違いと、macOS Terminal での挙動を整理する。

## 基本
- **ログインシェル**: ログイン時に 1 回だけ読み込まれる
- **対話シェル**: ターミナルを開くたびに読み込まれる

## 読み込みの役割
- `.zprofile`: ログインシェルで読み込まれる。PATH など環境変数の初期化向き。
- `.zshrc`: 対話シェルで読み込まれる。エイリアスや補完など操作性の設定向き。

## macOS Terminal での一般的な挙動
Terminal の設定が「デフォルトのログインシェル」の場合:
- **新しいウインドウ/タブ**を開くたびに `.zprofile` → `.zshrc` の順で読み込まれる
- 同じウインドウで `zsh` を再実行すると `.zshrc` のみ読み込まれる

## まとめ
- **環境の初期化**は `.zprofile`
- **操作性の設定**は `.zshrc`
