# 要件定義（RD）

## 目的 / 背景
- Mac のドットファイルを Git で履歴管理し、端末更新時に設定を安全・簡単に引き継ぐ。
- 変更理由（Why）と現行設定（What）を追跡できる状態を維持する。

## スコープ
### 対象（管理対象）
- シェル設定: `.zshrc`, `.zprofile`, `.bashrc`, `.bash_profile`, `.aliases`
- Git 設定: `.gitconfig`, `.gitignore_global`
- エディタ設定: `.vimrc`

### 対象外
- OS のシステム設定（UI 設定、システム環境設定）
- 端末固有の秘密情報（鍵・トークン・パスワード）
- Homebrew 等のパッケージ管理（本リポジトリに明示がないもの）

## ユースケース
- 新しい Mac にセットアップする
- 既存 Mac でドットファイルを更新し、履歴に残す
- トラブル時に直前の状態へ戻す

## 非機能要件 / 制約
- **秘密情報は含めない**（鍵・トークン・パスワード・実IP 等）
- Git で差分・履歴が追えること
- 適用はホームディレクトリのファイルと同期できること
- 端末更新時に短時間で復旧できること

## 成功条件
- 端末更新時、数分で主要ドットファイルが復元できる
- 変更理由と履歴が追跡できる
- 秘密情報を含まずに運用できる
