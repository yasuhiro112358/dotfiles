# ADR-0001: ドットファイルを Git 管理しシンボリックリンクで反映する

- Status: Accepted
- Date: 2025-12-21

## Context
- Mac のドットファイルを履歴管理し、端末更新時に再現可能にしたい。
- 変更理由や差分が追える運用が必要。

## Decision
- `~/dotfiles` 配下でドットファイルを Git 管理する。
- ホームディレクトリにはシンボリックリンクで反映する。
  - ※ リポジトリ配置場所は ADR-0003 により更新。

## Alternatives
1. **手動コピーで運用する**
   - 適用は簡単だが、差分・履歴が追いにくい。
2. **Dotfiles 管理ツール（例: stow / chezmoi）を導入する**
   - 自動化は進むが、学習コストと依存が増える。
3. **スクリプトで自動配置する**
   - 一部自動化は可能だが、運用負担と保守が発生する。

## Consequences
- リンク運用により即時反映・差分追跡が可能になる。
- 初回セットアップ時に既存ファイルの退避が必要。
- ツール依存が少なく、移行時の手順が単純。
