# Powerlevel10k テーマ設定再編集手順

## 目的
Powerlevel10kのプロンプト設定を後から変更・カスタマイズする方法を記載する。

## 前提
- Powerlevel10kが既にインストールされていること
- `.zshrc`で `ZSH_THEME="powerlevel10k/powerlevel10k"` が設定されていること

## 設定を再編集する方法

### 方法1: 設定ウィザードを再実行（推奨）

インタラクティブな設定ウィザードを使用して、視覚的に設定を変更できます。

```bash
p10k configure
```

このコマンドを実行すると：
- プロンプトのスタイルを選択できる
- 表示する情報（Git、Node、Dockerなど）を選択できる
- 色やアイコンをカスタマイズできる
- 設定は `~/.p10k.zsh` に自動保存される

### 方法2: 設定ファイルを直接編集

設定ファイルを直接編集して細かくカスタマイズすることも可能です。

```bash
vim ~/.p10k.zsh
```

または、`.zshrc`で定義されているエイリアスを使用：

```bash
zshconfig  # .zshrcを編集するエイリアス
```

設定ファイルの場所: `~/.p10k.zsh`

## 設定を反映させる方法

設定ファイルを編集した場合、以下のいずれかの方法で設定を反映させます：

### 方法1: `.zshrc`を再読み込み

```bash
source ~/.zshrc
```

### 方法2: エイリアスを使用

```bash
reload  # .zshrcで定義されているエイリアス
```

### 方法3: 新しいターミナルセッションを開く

ターミナルを再起動すると、自動的に新しい設定が読み込まれます。

## 設定ファイルの場所

- テーマ設定: `~/.zshrc`（`ZSH_THEME="powerlevel10k/powerlevel10k"`）
- カスタマイズ設定: `~/.p10k.zsh`（設定ウィザードで生成される）
- インスタントプロンプト: `~/.cache/p10k-instant-prompt-*.zsh`（自動生成）

## 注意事項

- `p10k configure` を実行すると、既存の `~/.p10k.zsh` が上書きされます
- 設定ファイルを直接編集する場合は、バックアップを取ることを推奨します
- 設定が反映されない場合は、`source ~/.zshrc` を実行してください
